# 🚁 Advanced Drone Control

## 📌 Project Overview

This project was completed as part of the India Space Lab Internship.

It demonstrates the fundamentals of autonomous drone control using Python simulations. The project includes PID-based altitude control, boat guidance, and a bonus figure-eight trajectory tracking simulation.

---

## ✨ Features

- PID altitude control simulation
- Interactive PID tuning using sliders
- Wind disturbance handling
- Boat guidance simulation
- Figure-eight trajectory tracking (Bonus Task)
- Python animation using Matplotlib

---

## 🛠 Technologies Used

- Python
- NumPy
- Matplotlib
- ipywidgets
- Google Colab
- Jupyter Notebook

## 📈 Simulations Included

### 1. PID Altitude Control
- Stable altitude control
- Wind disturbance analysis
- PID parameter tuning

### 2. Boat Guidance
- Path tracking
- Current disturbance handling
- Gain and damping tuning

### 3. Figure-Eight Trajectory (Bonus)
- Waypoint tracking
- PD controller
- Wind disturbance
- Animated trajectory

## 🚀 How to Run

1. Open the notebook in Google Colab or Jupyter Notebook.
2. Run all cells in order.
3. Adjust controller parameters using the sliders.
4. Observe the generated plots and animations.
